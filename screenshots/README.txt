Screenshot Guidelines:

1. Take screenshots of the following pages:
   - home.png (1200x800): Homepage showing the list of boards
   - discussion.png (1200x800): A board page showing topics
   - profile.png (1200x800): User profile page
   - topic.png (1200x800): Individual topic with posts

2. Screenshot requirements:
   - Resolution: 1200x800 pixels minimum
   - Format: PNG
   - Clear and well-lit interface
   - Include some sample content
   - Ensure no sensitive information is visible

3. Image naming:
   - Use lowercase
   - Use descriptive names
   - Use hyphens for spaces
   - Use .png extension
